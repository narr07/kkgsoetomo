🎨 ANIMATION TYPES (7 Pilihan)

  1. fadeIn .............. Fade dari 0 ke 1 opacity
  2. slideUp ............. Slide naik dari bawah
  3. slideDown ........... Slide turun dari atas
  4. slideLeft ........... Slide dari kanan ke kiri
  5. slideRight .......... Slide dari kiri ke kanan
  6. scaleIn ............. Scale dari 0.8 ke 1
  7. bounce .............. Bounce spring effect

  Gunakan:
  <AnimatedDiv animation="slideUp">...</AnimatedDiv>

🔧 COMMON PATTERNS

  Scroll-Triggered:
  <AnimatedDiv animation="slideUp">
    <div>Content</div>
  </AnimatedDiv>

  Hover Effect:
  <motion.button
    whileHover={{ scale: 1.05 }}
    whileTap={{ scale: 0.95 }}
  >
    Button
  </motion.button>

  Staggered List:
  {items.map((item, i) => (
    <AnimatedDiv delay={i * 0.1}>
      <div>{item}</div>
    </AnimatedDiv>
  ))}

  Continuous Loop:
  <motion.div
    animate={{ rotate: 360 }}
    transition={{ duration: 2, repeat: Infinity }}
  />

🌟 BEST PRACTICES

  ✅ DO:
  • Gunakan once: true untuk scroll animations
  • Keep animations short (0.3-0.6s)
  • Gunakan staggered delays untuk lists
  • Use easeOut untuk enter animations
  • Test di mobile devices

  ❌ DON'T:
  • Animate layout properties
  • Animate terlalu banyak elements
  • Use durasi terlalu lama (>1s)
  • Lupa 'use client' directive
  • Skip viewport optimization
