🔍 QUERIES USAGE
─────────────────────────────────────────────────────────────────────────

import { 
  allArticlesQuery,          // Fetch all articles
  articleBySlugQuery,        // Fetch single by slug
  relatedArticlesQuery       // Fetch related by category
} from '@/sanity/lib/queries'

List Page Query:
  allArticlesQuery
  → Returns: Array of articles with meta info
  → Used for: Initial load + search

Detail Page Queries:
  articleBySlugQuery (slug param)
  → Returns: Single article with full content
  
  relatedArticlesQuery (categoryId, currentId params)
  → Returns: Array of related articles (same category)

📋 DATA MODEL
─────────────────────────────────────────────────────────────────────────

Article interface:
  _id: string                    // Sanity ID
  title: string                  // Article title
  slug: { current: string }      // URL slug
  excerpt: string                // Summary
  author: {                      // Author info
    _id: string
    name: string
    image?: SanityImage
  }
  category: {                    // Category info
    _id: string
    title: string
    color?: string
  }
  publishedAt: string            // Pub date
  updatedAt?: string             // Update date
  image: SanityImage             // Featured image
  content: PortableTextBlock[]   // Rich content
  tags?: string[]                // Tags
  views?: number                 // View count
